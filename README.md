# BTC-Ledger-Web-App

BTC Ledger é um livro-razão digital para Bitcoin.

Criado para quem prefere controle à conveniência e soberania à dependência.

O app permite registrar compras, vendas e movimentações em satoshis, acompanhar o patrimônio em tempo real e analisar a performance da carteira com base no preço atual do Bitcoin — tudo isso sem backend, sem contas, sem coleta de dados.

Os dados ficam exclusivamente no dispositivo do usuário, respeitando um princípio antigo e imutável: quem controla as chaves, controla o destino.

BTC Ledger não é uma exchange.
Não é uma carteira.
É contabilidade limpa para quem leva Bitcoin a sério.
